# 🎓 Learning Management System (LMS)

A full-stack Learning Management System built with modern web technologies.  
This platform allows users to register, purchase courses, and access learning content seamlessly.

🌐 **Live Website:**  
https://learning-frontend-4bzg.onrender.com/

---

## 🚀 Features

- 🔐 User Authentication (JWT-based)
- 📚 Course Management
- 💳 Razorpay Payment Integration
- ☁️ Cloudinary Media Upload
- 📧 Email Support (SMTP)
- 🍪 Secure Cookie Handling
- 🛡️ Error Handling Middleware
- 🌍 CORS Enabled for Secure Frontend-Backend Communication

---

## 🏗️ Tech Stack

### 🔹 Frontend
- React (Vite)
- Axios
- Redux Toolkit
- Tailwind CSS
- React Router

### 🔹 Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Razorpay
- Cloudinary
- Nodemailer
- Morgan
- Cookie Parser
- CORS

### 🔹 Deployment
- Frontend → Render (Static Site)
- Backend → Render (Web Service)
- Database → MongoDB Atlas
